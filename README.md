# Embroidery Digitizer

testing embroidery design tools for those starting clothing brands

compatible with Brother/Babylock machines.

<img width="1512" height="763" alt="image" src="https://github.com/user-attachments/assets/3358f11a-9501-4ad1-9e39-7eafcb5ecc90" />

## Features

- Vector drawing tools (shapes, lines, paths)
- Image tracing support
- Multiple stitch types (fill, satin, running)
- PES file export
- Dark/Light mode
